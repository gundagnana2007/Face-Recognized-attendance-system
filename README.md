<p>SmartAttend is a contactless, automated attendance management solution that leverages Computer Vision and Deep Learning to streamline identity verification. By utilizing a Convolutional Neural Network (CNN) architecture, the system recognizes registered faces in real-time and logs attendance data into an automated database, eliminating manual roll calls and proxy attendance.</P>
<hr>
<h1>🚀 Key Features<h1>

Real-Time Detection: Instant face localization and recognition using live webcam feeds.<br>

High Accuracy: Powered by a CNN-based feature extraction model designed to handle variations in lighting and head poses.<br>

Automated Logging: Seamlessly records Student ID, Name, Date, and Timestamp into a CSV/Excel backend.<br>

Anti-Proxy Logic: Ensures unique identification through biometric landmarks, preventing students from marking attendance for others.<br>

Admin Dashboard: A lightweight Flask web interface for faculty to manage student rosters and export attendance reports.<br>
<hr>
<h1>🛠️ Tech Stack</h1>
<p>
Language: Python 3.12<br>

AI/Deep Learning: TensorFlow, Keras (CNN Architecture)<br>

Computer Vision: OpenCV (Image Pre-processing & Live Streaming)<br>

Web Framework: Flask (Backend & Dashboard)<br>

Data Handling: NumPy & Pandas (Matrix operations & CSV management)<br>
